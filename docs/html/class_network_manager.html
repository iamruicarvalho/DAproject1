◆ readFiles()

void NetworkManager::readFiles ( )

Lê os ficheiros stations.csv e network.csv e cria um grafo com as estações (vertices) e ligações (edges). Guardamos também as estações numa hash table para facilitar a procura de estações por id e nome. O id da estação é feito por auto-incrementação. Complexidade temporal: O(n+m), onde n é o número de estações e m é o número de ligações.

Returns
